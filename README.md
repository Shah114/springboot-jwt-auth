# Spring Boot JWT Authentication System

Backend authentication system built with:

- Java 17
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Spring Data JPA
- BCrypt Password Hashing

## Features

- User Registration
- User Login
- JWT Token Authentication
- Role-Based Authorization
- Protected API Endpoints

## Tech Stack

- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Maven

## Project Status

Learning project for backend development practice.
